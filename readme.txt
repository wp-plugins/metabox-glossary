Metabox Glossary allows you to set glossary items to a certain post and show it on a Widget

== Description ==

Metabox Glossary allows you to set glossary items to a certain post and allow you to show it on a Widget, also shows all Glossary items in a list or filtered from A to Z on the Front page..

Major features in Metabox Glossary include:

* Filtering of Glossary Items per Post
* Adds Glossary Metabox on post creation or Edit
* Show Items Selected when post is show on a Widget
* Shows all Glossary Items on Front Page on a list or filtered from A to Z
* Uses Ajax

This Plugin needs WP-Glossary Plugin to work, pls. install it first before activation

== Installation ==

Upload the Metabox Glossary plugin to your blog, Activate it, begin to use it.

1, 2, 3: You're done!
